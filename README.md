# School Health Hub

School Health Hub는 보건교사가 학생 건강검진, 소변검사, 결핵검진 운영을 준비하고 당일 진행 상황을 공유하며 종료 후 보고서까지 정리할 수 있도록 돕는 Vite + React + TypeScript 기반 Lite 웹앱입니다.

학생 이름과 명렬표 정보는 검진 운영센터 내부에서만 사용하며, 교사용 공유 화면에는 학생 전체 목록을 기본 노출하지 않습니다.

## 릴리스 버전

- 현재 릴리스 준비 버전: **v3.0.0**
- 저장소 이름: `school-health-check-scheduler`
- 제품명: **School Health Hub**

## 주요 기능

- 검사 유형 선택: 소변검사, 결핵검진, 일반 건강검진 세션 관리
- 시간표 자동배정: 학교 일과표, 업체 가능 시간, 수업 시간표, 장소 조건 반영
- 명렬표 관리: 세션 기준 학생 목록 업로드, 상태 변경, 메모 관리
- 검진 운영센터: 현재/다음 학급, 완료, 미도착, 지연 시간, 안내 문구, 운영 로그 관리
- 교사용 현황판: 리로스쿨 메시지로 공유할 수 있는 보기 전용 현황 화면
- 교무실/관리자 현황판: 전체 진행률, 미검 수, 최근 로그 요약 확인
- 운영 보고서: 관리자 보고용 문구 복사, 특이사항 기록, 인쇄/PDF 저장
- 학교 설정: 학교명, 기본 장소, 기본 시간, 안내 문구, 문의 연락처 저장
- 저장소 설정: localStorage 모드와 Supabase 실시간 모드 선택

## 메뉴 구조

1. 검사별 사용 안내
2. 대시보드
3. 검진 운영
4. 교사용 현황판
5. 교무실/관리자 현황판
6. 운영 보고서
7. 검사 조건
8. 학교 설정

10. 시간표 입력

12. 결과/출력
13. 실시간 검진 운영
14. 저장소 설정

## 사용 흐름

### 1. 검진 준비

1. 첫 화면에서 검사 유형을 선택합니다.
2. 학교 설정에서 학교명, 기본 검진 장소, 기본 시간, 안내 문구를 확인합니다.
3. 검진 세션을 생성하거나 기존 세션을 선택합니다.
4. 검사 조건과 시간표를 입력한 뒤 자동배정을 실행합니다.
5. 명렬표를 업로드해 세션 기준 학생 목록을 준비합니다.

### 2. 검진 당일

1. 검진 운영센터에서 현재 검사 학급과 다음 검사 학급을 지정합니다.
2. 완료, 미도착, 미도착 해제, 지연 시간, 운영 메모를 기록합니다.
3. 교사용 현황판 링크와 리로스쿨 발송 문구를 복사해 교직원 내부에 공유합니다.
4. 교무실/관리자 현황판에서 전체 진행률과 최근 운영 로그를 확인합니다.

### 3. 검진 종료

1. 미검 학생과 학급별 진행 현황을 확인합니다.
2. 운영 보고서에서 특이사항 및 개선 필요사항을 입력합니다.
3. 관리자 보고 문구를 복사하거나 보고서를 인쇄/PDF로 저장합니다.
4. 운영 로그와 보고서를 바탕으로 다음 검진 개선 사항을 정리합니다.

## 저장 방식

### localStorage 모드

- 기본 저장 방식입니다.
- 현재 브라우저에 검사 조건, 시간표, 세션, 명렬표, 운영상태, 로그가 저장됩니다.
- 브라우저 또는 기기를 바꾸면 데이터가 자동 공유되지 않습니다.

### Supabase 실시간 모드

- 환경변수 `VITE_SUPABASE_URL`, `VITE_SUPABASE_ANON_KEY`가 필요합니다.
- 세션, 학생 명렬표, 운영상태, 운영 로그를 Supabase와 연동합니다.
- Realtime이 가능하면 운영센터, 교사용 현황판, 관리자 현황판, 보고서가 자동 갱신됩니다.
- Supabase가 미설정이거나 연결에 실패하면 앱은 localStorage fallback으로 계속 동작합니다.

설정 방법은 [Supabase 설정 안내](docs/SUPABASE_SETUP.md)를 참고하세요.

## 리로스쿨 공유 방식

운영센터의 공유 패널에서 다음 링크와 문구를 복사할 수 있습니다.

- 교사용 링크: 수업 중 학생 이동 안내용 보기 전용 화면
- 교무실/관리자 링크: 전체 진행률과 최근 로그 확인용 화면
- 리로스쿨 발송문: 교사용 현황판 링크와 검진 안내 문구 포함

링크는 교직원 내부 공유용입니다. 학생 개인정보 보호를 위해 외부 공유를 금지합니다.

## 운영 보고서 인쇄 및 PDF 저장

운영 보고서 화면에서 검진 종료 후 관리자 보고 및 보관용 보고서를 출력할 수 있습니다.

1. 사이드바에서 운영 보고서를 엽니다.
2. 특이사항 및 개선 필요사항을 입력하고 저장합니다.
3. 인쇄하기 버튼을 눌러 브라우저 인쇄 창을 엽니다.
4. 종이 출력이 필요하면 프린터를 선택하고, PDF 보관이 필요하면 대상을 PDF 저장으로 선택합니다.
5. 화면에 표시되는 추천 파일명을 참고해 저장합니다. 예: `2026-06-26_결핵검진_운영보고서.pdf`

## 개인정보 및 운영 주의사항

- 학생 이름과 명렬표는 검진 운영 확인용으로만 사용합니다.
- 검사 결과, 질병명, 상담 내용, 주민등록번호 등 민감정보는 입력하지 않습니다.
- 교사용 현황판에는 학생 이름 전체 목록을 기본 표시하지 않습니다.
- 공유 링크는 교직원 내부 안내용으로만 사용합니다.
- 공개 배포 전 학교별 개인정보 처리 기준을 반드시 확인하세요.

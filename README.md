# 검진·검사 시간표 자동배정 도우미

학교 보건 업무에서 소변검사와 결핵검진 시간표를 빠르게 만들기 위한 내부 업무용 React 앱입니다.

## 핵심 원칙

- 학생 이름, 학번, 검사 결과, 질병명 등 개인정보를 입력하지 않습니다.
- 소변검사는 실제 방문 가능한 교실·장소 기준으로 배정합니다.
- 결핵검진은 호출 가능한 학급·수업단위 기준으로 호출 시간과 검진 예상 시간을 계산합니다.
- 데이터는 브라우저 `localStorage`에 저장됩니다.

## 주요 기능

- 첫 화면에서 검사 유형 선택
  - 소변검사: 방문형 검사, 교실·장소 기준
  - 결핵검진: 호출형 검진, 검진 장소 이동 기준
- 학교 일과 시간표 preset 설정
- 업체 검사 가능 시간과 단위당 소요시간 반영
- 검사 불가/주의 키워드 관리
- 방문 장소 또는 호출 단위 목록 관리
- 컴시간알리미 엑셀 업로드
- 공통 시간표 서식 `.xlsx` / `.csv` 다운로드 및 업로드
- 선택과목 분반 수동 확인 목록 관리
- 자동배정 및 수동 조정
- 결과표 CSV 다운로드, 인쇄, 안내 문구 복사
- JSON 백업 및 복원
- 연도별 검사 템플릿 저장 및 복사

## 자동배정 기준

자동배정 가능 시간은 다음 조건의 교집합으로 계산합니다.

- 학교 일과표상 배정 가능 시간
- 업체 검사 가능 시작/종료 시간
- 사용자가 제외하지 않은 시간
- 검사 유형별 단위당 예상 소요시간
- 결핵검진의 경우 이동 소요시간

결핵검진은 `호출 시간 = 검진 예상 시간 - 이동 소요시간`으로 계산합니다.

```

## 빌드

```

- React

- Vite
- localStorage

## 개인정보 주의

이 저장소에는 학생 개인정보, 실제 학생 명단, 실제 검사 결과 데이터를 포함하지 않습니다. 포함된 데이터는 앱 동작 확인용 샘플 시간표입니다.

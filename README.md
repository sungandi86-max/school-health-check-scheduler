# 검진·검사 시간표 자동배정 도우미

학교 보건 업무에서 소변검사와 결핵검진 시간표를 빠르게 만들기 위한 내부 업무용 React 앱입니다.

## 핵심 원칙

- 학생 이름, 학번, 검사 결과, 질병명 등 개인정보를 입력하지 않습니다.
- 소변검사는 실제 방문 가능한 교실·장소 기준으로 배정합니다.
- 결핵검진은 호출 가능한 학급·수업단위 기준으로 호출 시간과 검진 예상 시간을 계산합니다.
- 데이터는 브라우저 `localStorage`에 저장됩니다.

## 주요 기능

- 첫 화면에서 검사 유형 선택
  - 소변검사: 방문형 검사, 교실·장소 기준
  - 결핵검진: 호출형 검진, 검진 장소 이동 기준
- 학교 일과 시간표 preset 설정
- 업체 검사 가능 시간과 단위당 소요시간 반영
- 검사 불가/주의 키워드 관리
- 방문 장소 또는 호출 단위 목록 관리
- 컴시간알리미 엑셀 업로드
- 공통 시간표 서식 `.xlsx` / `.csv` 다운로드 및 업로드
- 선택과목 분반 수동 확인 목록 관리
- 자동배정 및 수동 조정
- 결과표 CSV 다운로드, 인쇄, 안내 문구 복사
- JSON 백업 및 복원
- 연도별 검사 템플릿 저장 및 복사

## 자동배정 기준

자동배정 가능 시간은 다음 조건의 교집합으로 계산합니다.

- 학교 일과표상 배정 가능 시간
- 업체 검사 가능 시작/종료 시간
- 사용자가 제외하지 않은 시간
- 검사 유형별 단위당 예상 소요시간
- 결핵검진의 경우 이동 소요시간

결핵검진은 `호출 시간 = 검진 예상 시간 - 이동 소요시간`으로 계산합니다.

## 개인정보 주의

이 저장소에는 학생 개인정보, 실제 학생 명단, 실제 검사 결과 데이터를 포함하지 않습니다. 포함된 데이터는 앱 동작 확인용 샘플 시간표입니다.

## 실시간 검진 운영 MVP

이번 버전에는 `실시간 검진 운영` 메뉴가 추가되었습니다.

- 보건교사용 관리 화면: 현재 검사 반, 다음 반, 미도착 반, 지연 시간 입력
- 명렬표 업로드: 학생별 완료/결석/조퇴/지각/추후검진 체크
- 교사용 모바일/PC 현황판: 현재 검사 반, 다음 반, 미도착 반, 지연 여부 확인
- 교무실 큰 화면: 교무실 PC 또는 모니터용 진행 현황판
- 관리자 구상안: 고교학점제 대응 검진 운영 개선 보고 문안

현재 MVP는 브라우저 `localStorage` 기반입니다. 여러 기기에서 동시에 공유하려면 다음 단계에서 Supabase 또는 Firebase 실시간 DB 연동이 필요합니다.

## School Health Hub

이 저장소의 현재 이름은 `school-health-check-scheduler`이지만, 장기 제품명은 **School Health Hub**로 사용합니다.

School Health Hub는 기존 학생건강검진 스케줄러를 기반으로 다음 기능까지 확장하는 학교 보건 업무 허브입니다.

- 검진 시간표 자동배정
- 공통 건강검진 모듈
- 명렬표 업로드 및 학생 상태 관리
- 검진 운영센터
- 교사용 대시보드
- 실시간 동기화
- 관리자 대시보드
- 운영 로그 및 보고서

## 버전 관리 정책

이 프로젝트는 GitHub Releases와 SemVer 형식을 기준으로 버전을 관리합니다.

- `v1.0.0`: 기존 학생건강검진 스케줄러 기준 버전
- `v2.x.x-alpha`: 공통 건강검진 구조와 내부 데이터 모델을 정리하는 초기 확장 단계
- `v2.x.x-beta`: 검진 운영센터, 교사용 대시보드, 실시간 동기화 등 현장 운영 기능 검증 단계
- `v3.0.0`: School Health Hub 첫 정식 버전

릴리스 전에는 최소한 다음 조건을 확인합니다.

- `npm run build` 통과
- 기존 소변검사/결핵검진 스케줄러 기능 유지
- 개인정보가 저장소 예시, 로그, 문서에 포함되지 않음
- 변경 내용이 `docs/CHANGELOG.md`에 기록됨

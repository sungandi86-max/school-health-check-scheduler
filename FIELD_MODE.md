# Field Mode

문서 시작점: [PROJECT.md](PROJECT.md)

## 목적

Field Mode(현장 모드)는
학교에서 진행되는 별도검사(결핵검진, 소변검사 등)의
현장 운영을 디지털화하기 위한 전용 화면입니다.

특정 사용자(강사)용 화면이 아니라

검진 현장에 있는 사람이 사용하는 운영 화면입니다.

---

# 핵심 개념

운영 준비

↓

운영 계획

↓

Field Mode (현장 운영)

↓

현황 공유

↓

운영 결과

Field Mode는
운영 준비와 현황 공유를 연결하는 핵심 단계입니다.

---

# 지원 검사

현재 지원

- 결핵검진
- 소변검사

향후 확장 가능

- 구강검진
- 시력검사
- 예방접종
- 기타 학교 별도검사

---

# 결핵검진

현장에서는

- 현재 학급 확인
- 학생 명렬표 확인
- 검진 완료 체크
- 결석
- 미검진 확인
- 다음 학급 이동

을 수행합니다.

체크 결과는 운영센터와 현황판에 실시간 반영됩니다.

---

# 소변검사

현장에서는

- 현재 학급 확인
- 검체 제출 확인
- 미제출
- 결석
- 확인 필요

를 체크합니다.

보건교사는 운영센터에서
소변검사의 진행률을 실시간 확인할 수 있습니다.

---

# UI 원칙

Field Mode는

빠른 체크가 목적입니다.

원칙

- 큰 글씨
- 큰 버튼
- 최소 입력
- 최소 스크롤
- 터치 중심
- 오입력 방지

복잡한 설정은 포함하지 않습니다.

---

# 기기별 역할

PC

- 운영 준비
- 시간표 생성
- 운영 계획
- 보고서

태블릿

- 현장 체크
- 학생 상태 기록

모바일

- 진행 상황 확인
- 운영 모니터링
- 긴급 수정

Display

- 현재 학급
- 다음 학급
- 진행률
- 안내

---

# 실시간 구조

Field Mode

↓

Storage

(localStorage 또는 Supabase)

↓

운영센터

↓

교사용 현황판

↓

관리자 현황판

↓

Display

모든 화면은 동일한 운영 데이터를 공유합니다.

---

# 개인정보 원칙

Field Mode에서는
운영에 필요한 최소 학생 정보만 표시합니다.

공용 현황판(Display, 관리자용)은
학생 이름과 학번을 기본 표시하지 않습니다.

# 설계 철학

Field Mode는

"검진 현장에서 종이를 없애는 것"

이 목적이 아닙니다.

목표는

학교 별도검사의 진행 상황을
실시간으로 공유하고
보건교사가 운영 전체를 한눈에 관리할 수 있도록 만드는 것입니다.

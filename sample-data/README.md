# QA and Demo Sample Data

이 폴더의 파일은 학교 별도검사 운영 도우미 QA와 시연을 위한 가상 데이터입니다.

- 실제 학생 이름, 학번, 건강 정보가 아닙니다.
- QA, 시연, 기능 확인용으로만 사용합니다.
- 실제 운영 전에는 반드시 학교의 실제 명렬표와 운영 조건으로 교체해야 합니다.
- 공용 화면 점검 시 학생 이름과 번호가 교사용/관리자/Display 화면에 노출되지 않는지 확인합니다.

## 파일

- `qa-demo-roster.csv`: 2학년 1반~12반, 3학년 1반~12반 가상 명렬표
- `qa-demo-scenario.md`: 완료, 미완료, 미도착 학급, 결석 학생, 지연 상황을 만드는 시연 절차

## 사용 방법

1. 앱에서 테스트용 세션을 만듭니다. 예: `2027 결핵검진 QA`.
2. 운영센터에서 `qa-demo-roster.csv`를 명렬표로 업로드합니다.
3. `qa-demo-scenario.md`의 순서대로 현재 학급, 완료, 미도착, 결석, 지연 상태를 입력합니다.
4. 교사용 현황판, 관리자 현황판, Display Portrait, Display Landscape, Tablet Mode, 운영 보고서를 차례로 확인합니다.

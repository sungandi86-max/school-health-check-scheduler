# 변경 이력

이 문서는 Keep a Changelog 형식을 참고하여 School Health Hub / 학교 별도검사 운영 도우미의 주요 변경 사항을 한국어로 기록합니다.

## [Unreleased]

### 예정
- v1.0.0 MVP 릴리스 이후 현장 피드백을 반영합니다.

## [v1.0.0] - MVP 릴리스

### 추가
- Home 첫 화면에서 소변검사 운영 계획, 결핵검진 실시간 운영, 운영 현황판을 선택할 수 있게 정리했습니다.
- 최근 작업 카드, 새 검진 시작, 기존 세션 선택을 포함한 Session 관리 UX를 정리했습니다.
- 운영센터에서 현재 학급, 다음 학급, 진행률, 남은 학급, 예상 종료, 미도착, 지연 상태를 관리할 수 있게 했습니다.
- 교사용 현황판에서 현재/다음/다다음 학급과 미도착·지연 상태를 확인할 수 있게 했습니다.
- 관리자 현황판에서 전체 진행률, 학급별 상태, 미도착·지연, 최근 운영 로그를 확인할 수 있게 했습니다.
- Display Mode를 보건실 세로형과 교무실·관리자 가로형 사이니지 화면으로 분리했습니다.
- Tablet Mode를 검진장 검사자가 사용할 수 있는 큰 버튼 중심 화면으로 정리했습니다.
- 운영 보고서에서 운영 요약, 특이사항, 개선 메모, 인쇄/PDF 저장 흐름을 제공합니다.

### 변경
- 사이드바 메뉴를 보건교사의 업무 흐름 중심으로 재구성했습니다.
- Home 화면에서 검사 유형 선택과 최근 작업이 먼저 보이도록 정보 우선순위를 조정했습니다.
- Home의 운영 현황판 이동 경로를 검사 조건 입력이 아니라 현황판 화면으로 정리했습니다.
- 모바일·태블릿 반응형 레이아웃과 현장 예외 상황 표시를 정리했습니다.
- 완료, 진행 중, 미도착, 결석, 대기 상태의 색상 규칙을 통일했습니다.
- Session 관리 화면을 최근 작업, 새 검진 시작, 기존 세션 선택 흐름으로 정리했습니다.
- Display Mode와 Tablet Mode를 실제 학교 기기 사용 상황에 맞게 읽기 쉬운 화면으로 조정했습니다.

### 비고
- v1.0.0 MVP는 결핵검진·소변검사 운영을 중심으로 한 학교 별도검사 운영 도우미의 첫 릴리스입니다.
- 저장 구조, Repository, DataService, StorageAdapter, 계산 로직은 기존 구조를 유지합니다.

# 변경 이력

이 문서는 Keep a Changelog 형식을 참고하여 School Health Hub 프로젝트의 주요 변경 사항을 한국어로 기록합니다.

- 교사용 대시보드

## [v2.0.0-alpha] - 공통 건강검진 구조 리팩터링 준비

### 추가
- School Health Hub 확장을 위한 공통 건강검진 타입 구조를 준비했습니다.
- `HealthCheckType` 기반으로 소변검사, 결핵검진, 일반 건강검진, 기타검진을 구분할 수 있게 했습니다.
- 향후 운영센터, 명렬표, 보고서가 같은 검사 종류 구조를 사용할 수 있도록 기반을 마련했습니다.

### 변경
- 기존 저장소 이름은 `school-health-check-scheduler`를 유지하되, 제품명은 School Health Hub로 확장합니다.
- 기존 스케줄러 중심 구조를 장기적으로 공통 건강검진 모듈로 전환하는 방향을 정의했습니다.

### 비고
- alpha 단계에서는 기존 소변검사 및 결핵검진 스케줄러 기능을 유지하는 것을 우선합니다.

## [v1.0.0] - 기존 검진 스케줄러

### 추가
- 학생건강검진 스케줄러의 기존 핵심 기능을 기준 버전으로 정의했습니다.
- 소변검사 방문형 자동배정 기능을 제공합니다.
- 결핵검진 이동형 자동배정 기능을 제공합니다.
- 학교 일과표, 업체 검사 가능 시간, 교시, 장소 조건을 반영합니다.
- CSV 다운로드, 인쇄, 안내 문구 복사, JSON 백업 기능을 제공합니다.

### 비고
- v1.0.0은 School Health Hub로 확장하기 전의 안정 기준선입니다.

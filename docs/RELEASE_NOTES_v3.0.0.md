# School Health Hub v3.0.0 릴리스 노트 초안

## 릴리스 개요

School Health Hub v3.0.0은 단순 검진 시간표 자동배정 도구에서 검진 준비, 당일 운영, 교직원 공유, 종료 후 보고서 작성까지 이어지는 보건교사용 운영 허브로 확장한 정식 릴리스 후보입니다.

이번 릴리스는 소변검사와 결핵검진 자동배정 기능을 유지하면서 세션 기반 운영센터, 명렬표 관리, Supabase Realtime 연동, 공유용 현황판, 인쇄용 운영 보고서를 포함합니다.

## 주요 기능

- 검사 유형별 시간표 자동배정
  - 소변검사: 검사팀 방문형, 실제 수업 장소 기준 배정
  - 결핵검진: 학생 호출형, 이동 소요 시간과 학년별 시간 구간 반영
- 세션 관리
  - 검진 종류, 날짜, 대상 학년, 장소, 진행 상태 관리
  - localStorage 및 Supabase 저장 모드 지원
- 명렬표 관리
  - 세션 기준 학생 목록 업로드
  - 학생별 검진 상태와 메모 관리
  - 학급별 학생 목록 및 상태 요약
- 검진 운영센터
  - 현재 검사 학급, 다음 검사 학급, 완료 학급, 미도착 학급 관리
  - 지연 시간, 안내 문구, 운영 메모, 운영 로그 기록
- 공유 화면
  - 교사용 현황판: 수업 중 학생 이동 안내용 보기 전용 화면
  - 교무실/관리자 현황판: 전체 진행률과 최근 로그 확인
  - 리로스쿨 발송문 자동 생성 및 복사
- 운영 보고서
  - 학생 검진 현황, 학급 진행 현황, 운영 로그 요약
  - 특이사항 및 개선 필요사항 기록
  - A4 인쇄 및 PDF 저장 안내
- 학교 설정
  - 학교명, 기본 장소, 기본 시간, 안내 문구, 문의처 저장
- 저장소 설정
  - localStorage 모드
  - Supabase 실시간 모드 및 연결 실패 시 fallback

## 사용 시나리오

### 검진 준비

1. 학교 설정을 확인합니다.
2. 검진 세션을 생성합니다.
3. 검사 조건과 시간표를 입력합니다.
4. 자동배정을 실행합니다.
5. 명렬표를 업로드합니다.

### 검진 당일

1. 운영센터에서 현재/다음 학급을 지정합니다.
2. 완료, 미도착, 지연 시간을 기록합니다.
3. 교사용 현황판 링크와 리로스쿨 문구를 공유합니다.
4. 관리자 현황판에서 전체 진행률과 최근 로그를 확인합니다.

### 검진 종료

1. 미검 학생과 미도착 기록을 확인합니다.
2. 운영 보고서에 특이사항과 개선 필요사항을 입력합니다.
3. 보고서를 인쇄하거나 PDF로 저장합니다.
4. 다음 검진 개선 사항을 정리합니다.

## 알려진 제한사항

- 로그인/회원가입은 아직 제공하지 않습니다.
- 역할 기반 화면 모드는 안내와 UI 구분 중심이며, 강제 접근 제어는 추후 도입 예정입니다.
- 학교 설정은 현재 localStorage에 저장되며 Supabase school_settings 테이블 연동은 후속 작업입니다.
- Supabase Realtime은 환경변수와 테이블 설정이 올바른 경우에만 동작합니다.
- PDF 저장은 별도 PDF 라이브러리가 아니라 브라우저 인쇄 기능을 사용합니다.
- 명렬표 파일은 학교별 서식 차이에 따라 열 이름 확인이 필요할 수 있습니다.

# School Health Hub Product Roadmap

이 문서는 School Health Hub를 장기 학교 보건 플랫폼으로 발전시키기 위한 제품 로드맵입니다.

## Milestone 1: v3.0 학생건강검진 플랫폼

목표: 검진 준비, 당일 운영, 공유, 보고서까지 하나의 흐름으로 연결합니다.

포함 범위:

- 검진 스케줄러
- 세션 관리
- 명렬표 관리
- 검진 운영센터
- 교사용 현황판
- 관리자 현황판
- 운영 로그
- 운영 보고서
- 운영 분석 기반 구조
- Supabase Realtime

완료 기준:

- localStorage 모드와 Supabase 모드 모두 빌드 통과
- 세션, 학생, 운영상태, 로그가 저장 방식에 따라 동작
- 교사용 공유 링크와 관리자 현황판이 운영센터 변경을 반영
- 운영 보고서 인쇄/PDF 저장 가능

## Milestone 2: v4.0 협업 플랫폼

목표: 보건교사 혼자 쓰는 도구에서 교직원이 역할별로 함께 쓰는 협업 플랫폼으로 확장합니다.

주요 기능:

- 역할별 화면 정교화
- 권한 관리
- 교사용 공유 링크 개선
- 학교 설정 고도화
- 알림 시스템
- 교사별 확인 여부 또는 안내 수신 여부 기록

검토 사항:

- Supabase Auth 또는 PIN 기반 접근 제한
- 공유 링크 만료 시간
- 보기 전용 화면과 수정 가능 화면의 명확한 분리

## Milestone 3: v5.0 멀티스쿨

목표: 여러 학교가 각자의 데이터를 분리해 사용할 수 있는 구조를 만듭니다.

주요 기능:

- 학교별 데이터 분리
- 학교별 설정
- 학교별 관리자
- 교육청 확장 가능 구조
- 학교별 템플릿과 기본 운영 규칙

검토 사항:

- 학교 ID 기반 Row Level Security
- 학교 관리자 초대 방식
- 교육청 단위 통계 제공 가능성

## Milestone 4: v6.0 AI 운영 지원

목표: 운영 중 판단을 돕는 보조 기능을 제공합니다.

주요 기능:

- 검진 순서 추천
- 미도착 학급 대응 제안
- 수행평가 충돌 안내
- 조퇴/지각 학생 관리 지원
- 운영 보고서 자동 작성
- 이전 검진 로그 기반 개선 제안

검토 사항:

- 개인정보를 최소화한 AI 입력 구조
- 민감정보 입력 방지 장치
- 보건교사가 최종 판단하는 human-in-the-loop 방식

## Milestone 5: v7.0 학교 보건 통합 플랫폼

목표: 검진 운영을 넘어 학교 보건실 업무 전반을 연결합니다.

주요 기능:

- 학생건강검진
- 보건실 방문 관리
- 응급환자 관리
- 감염병 관리
- 투약 관리
- 보건교육
- 통계 및 분석
- 관리자 보고 자료 자동화

장기 방향:

- 학교 보건 업무의 일상 기록과 긴급 대응을 모두 지원
- 교직원 내부 공유와 개인정보 보호를 동시에 만족
- 학교별 운영 차이를 설정과 템플릿으로 흡수

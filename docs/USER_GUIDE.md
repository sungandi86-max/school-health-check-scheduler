# User Guide

School Health Hub를 처음 사용하는 보건교사를 위한 기본 사용 안내입니다.
이 앱은 검진 준비, 검진 당일 운영, 검진 종료 후 보고서 정리를 한 흐름으로 이어 주는 Lite 웹앱입니다.

## 처음 시작하기

1. 첫 화면에서 검사 유형을 선택합니다.
2. 검진 세션을 만듭니다.
3. 검사 조건과 시간표를 입력합니다.
4. 자동배정을 실행합니다.
5. 명렬표를 업로드합니다.
6. 검진 당일 운영센터에서 현재 학급과 다음 학급을 관리합니다.
7. 교사용 현황판 링크를 공유합니다.
8. 검진 종료 후 운영 보고서를 확인합니다.

첫 실행 안내에서 `다시 보지 않기`를 누르면 이 브라우저의 localStorage에 확인 여부가 저장됩니다.
저장 키는 `schoolHealthHub.onboarding.dismissed`입니다.

## 검진 준비 방법

### 1. 검진 세션 만들기

세션은 검진일, 검사 종류, 대상 학년, 검진 장소를 묶는 기준입니다.
세션을 먼저 만들면 이후 명렬표, 운영상태, 운영 로그를 같은 기준으로 관리할 수 있습니다.

### 2. 검사 종류 선택

지원하는 검사 유형은 다음과 같습니다.

- 소변검사: 검사팀이 교실 또는 실제 수업 장소로 방문하는 방식
- 결핵검진: 학생들이 검진 장소로 이동하는 호출형 방식
- 학교 별도검사: 결핵검진·소변검사 운영을 위한 확장 유형

### 3. 시간표 생성

검사 조건, 학교 일과표, 업체 검사 가능 시간, 수업 시간표를 입력한 뒤 `검사 시간표 자동배정하기`를 실행합니다.
시간표를 입력하기만 하면 결과가 생성되지 않으므로 반드시 자동배정 버튼을 눌러야 합니다.

### 4. 명렬표 업로드

명렬표를 업로드하면 학생별 완료, 미검, 결석, 조퇴, 지각, 추후검진 상태를 체크할 수 있습니다.
학생 개인정보는 보건교사용 운영 화면에서만 사용하고, 교사용 공유 화면에는 학생 전체 목록을 기본 노출하지 않습니다.

## 검진 당일 운영 방법

1. 검진 운영센터를 엽니다.
2. 현재 검사 학급을 지정합니다.
3. 완료된 학급은 `완료 처리`를 누릅니다.
4. 학생이 오지 않은 학급은 `미도착 표시`를 누릅니다.
5. 지연 시간이 있으면 지연 시간을 입력합니다.
6. 운영 메모와 운영 로그를 남깁니다.

운영 데이터가 없을 때는 먼저 운영센터에서 현재 검사반을 지정해 주세요.

## 교사용 링크 공유 방법

운영센터의 공유 영역에서 다음 링크와 문구를 복사할 수 있습니다.

- 교사용 현황판 링크: 수업 중 학생 이동 안내용
- 교무실/관리자 현황판 링크: 전체 진행률 확인용
- 리로스쿨 발송문: 교사용 링크와 안내 문구를 포함한 공유 메시지

교사용 현황판은 학급 단위 중심으로 표시되며 학생 이름 전체 목록은 기본 표시하지 않습니다.
링크는 교직원 내부 공유용으로만 사용하세요.

## 검진 종료 후 보고서 확인 방법

1. 운영 보고서 화면을 엽니다.
2. 검진 기본 정보, 학생 검진 현황, 학급 진행 현황, 운영 로그 요약을 확인합니다.
3. 특이사항 및 개선 필요사항을 입력합니다.
4. 인쇄하기 버튼으로 출력하거나 PDF 저장을 선택합니다.

## localStorage 모드와 Supabase 모드 차이

### localStorage 모드

- 기본 저장 방식입니다.
- 현재 브라우저에 데이터가 저장됩니다.
- 다른 기기와 실시간 공유되지 않습니다.
- Supabase 환경변수가 없어도 사용할 수 있습니다.

### Supabase 모드

- 세션, 명렬표, 운영상태, 운영 로그를 Supabase와 연동합니다.
- Realtime 설정이 되어 있으면 운영센터 변경 사항이 교사용/관리자 현황판에 반영됩니다.
- Supabase 연결 실패 시 앱은 localStorage fallback으로 계속 동작합니다.

## 문제 상황별 안내

- 세션이 없을 때: 먼저 검진 세션을 만들어 주세요.
- 명렬표가 없을 때: 명렬표를 업로드하면 학생별 체크가 가능합니다.
- 운영 데이터가 없을 때: 운영센터에서 현재 검사반을 지정해 주세요.
- 결과가 안 나올 때: 시간표 입력 후 `검사 시간표 자동배정하기`를 실행했는지 확인해 주세요.
